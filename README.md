# Portfolio

A professional one-page HTML portfolio website showcasing personal profile, about section, services, projects, and contact information.

## Overview

This is a modern, responsive portfolio website built with HTML and CSS. It features a clean design with a navigation bar, hero section, and about me section, perfect for freelancers and professionals looking to showcase their work and skills.

## Features

- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Navigation Bar**: Easy-to-use navigation menu with links to different sections
  - HOME
  - ABOUT
  - SERVICES
  - PROJECTS
  - CONTACT
- **Hero Section**: Eye-catching landing section with call-to-action button ("Let's begin")
- **About Section**: "My Story" section with personal introduction
- **Contact Information**: Phone number displayed in the header
- **Modern Styling**: Clean and professional CSS styling

## File Structure

```
portfilo.html      - Main portfolio page
style.css          - Stylesheet for all pages

index.html         - Home page variant

```

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - can be opened directly in a browser

### Installation

1. Clone or download the repository
2. Navigate to the project directory
3. Open `portfilo.html` in your web browser

```bash
# Simply open the file
start portfilo.html
```

## Usage

The portfolio is fully functional as-is. You can customize it by:

1. **Update Content**: Edit the HTML to add your own information, name, and details
2. **Change Images**: Replace image URLs with your own
3. **Modify Styles**: Update `style.css` to match your personal branding
4. **Add Sections**: Extend the page with additional sections as needed

## Customization Guide

### Edit Personal Information

Open `portfilo.html` and update:
- Title in `<title>` tag
- "Hello friend!" greeting with your name
- "I'm available for freelance work" with your own tagline
- Phone number in the contact info

### Update Images

Replace image URLs in the `<img>` tags with your own images:
- Profile photo (hero section)
- About section images
- Any other visual elements

### Brand Your Name

Change the "First" text in the navbar to your name or business name
